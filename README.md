## Instalacja

1. Pobierz `BLT_5.2.4_RoT_Dragon_v*.zip` z linku powyżej
2. Rozpakuj zawartość do folderu `Modules/` w katalogu gry:
   ```
   ...\Mount & Blade II Bannerlord\Modules\
   ```
3. Pobierz **MakeBltGreatAgain** — kliknij **Code → Download ZIP** lub sklonuj repo
4. Wrzuć folder `MakeBltGreatAgain` do tego samego `Modules/`
5. Uruchom grę i włącz oba mody w launcherze

## Co dodaje ten mod

### Strike Powers (aktywne)
| Moc | Efekt |
|-----|-------|
| Poison Strike | Zatruwa cel, zadaje obrażenia w czasie |
| Burning Strike | Podpala cel |
| Frost Strike | Spowalnia i mrozi cel |
| Vampirism Strike | Leczy atakującego za część zadanych obrażeń |
| Chain Lightning | Elektryczność skacze do pobliskich wrogów |
| Bleed Strike | Powoduje krwawienie |
| Jump Attack | Skacze w kierunku celu i uderza |
| Kick | Odpycha cel przy trafieniu |

### Aura Powers (pasywne — pobliskie jednostki)
| Moc | Efekt |
|-----|-------|
| Heal Aura | Leczy pobliskich sojuszników |
| Damage Aura | Zadaje obrażenia pobliskim wrogom |
| Curse Aura | Osłabia pobliskich wrogów |
| Buff Aura | Wzmacnia pobliskich sojuszników |
| Fear Aura | Powoduje ucieczkę pobliskich wrogów |
| Slow Aura | Spowalnia pobliskich wrogów |
| Weakness Aura | Redukuje pancerz/obrażenia pobliskich wrogów |
| Battle Cry Aura | Podnosi morale pobliskich sojuszników |
| Commander Aura | Silna aura — do odblokowania przez channel points |

### Survival Powers (pasywne — bohater)
| Moc | Efekt |
|-----|-------|
| Berserk | Zwiększa obrażenia gdy HP jest niskie |
| Last Stand | Silny bonus gdy bohater bliski śmierci |
| Blood Rage | Kumuluje bonus do obrażeń z każdym killem |
| Vengeance | Zwraca część otrzymanych obrażeń atakującemu |
| Absorb Health | Pochłania obrażenia i zamienia na zdrowie |
| Taunt | Zmusza pobliskich wrogów do atakowania bohatera |

### Support & Special Powers
| Moc | Efekt |
|-----|-------|
| War Banner | Sojusznicy w zasięgu dostają bonus do szybkości |
| Mark Target | Wrogowie w pobliżu — sojusznicy zadają im więcej obrażeń |
| Rallying Cry | Leczenie sojuszników gdy HP bohatera spadnie poniżej progu |
| Shadowstep | Teleportuje bohatera za najbliższego wroga |
| Stealth | Bohater staje się niemożliwy do namierzenia |

Wszystkie moce obsługują: **czas trwania**, **liczba stacków**, **efekty cząsteczkowe** i **kolor konturu** bohatera.
